What is grid based design
Grid based design is a website design method, you devide the screen into 12 columns and use them as a guide to know where to place your content
What is all the hype about responsive webpages
Today we have so many different devices that have web browsing capability and they are all diffrent sizes and have diffrent display requirements, making your web page "responsive" will allow it to be displayed well on diffrent devices
Semantic Structure
this is using the html tags to organise the information on your page according to their prominence. For example you would use the h1 tag for the title of your webpage and then down to h2 for the heading of the first article and the h3 for any headings within that article ect.
