What is 'scope', and how does it work in javascript?

The 'scope' of an element in Javascript defines where it can be used in the code. A 'global' variable can be used anywhere in the code, but a local variable is declared inside a function and can only be accessed inside the function.

In the javascript language, describe all the use cases for each of:

Parenthesis ()

  • To execute a function e.g. thisFunction()
  • Pass the arguments into a function e.g thisFunction(arg)
  • To define the conditions for a loop e.g. for(i=0;i<10;i++)
Square brackets []
  • To define an array e.g. var arr = [];
  • To reference an item in an array e.g. arr[1]
Braces {}
  • To define the code in a function e.g. thisFunction(){ do this code }
  • To define an object e.g. var someObject = {}
Single quotes ''
  • To define a string e.g. var aString = 'hello'
  • To indicate speech marks inside a string e.g. var aString = "the banana says 'hello'"
  • To define the properties in an object someObject = {'name': 'Bob Belcher'}
Double quotes ""
  • To define a string e.g. var aString = "hello"
